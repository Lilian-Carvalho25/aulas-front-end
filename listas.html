Trabalhando com listas

Lista é um tipo de conteúdo baseado em uma relação de itens.

Lista não ordenada

Relação de itens em que a ordem não importa.

Exemplos:

  • Café
  • Açucar
  • Chocolate
  • Farinha de trigo

Lista ordenada

Relação de itens em que a ordem importa.

Exemplos:

  1. Briefing
  2. Proposta comercial
  3. Wireframe/sistemap
  4. Layout/design

Outros níveis de lista

  • Front-end
    • HTML
    • CSS
    • JS
  • Back-end
    1. PHP
    2. SQL
    3. Node.js
    4. APIs
